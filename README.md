# Forja de Cordel

**Mapa de Significado → Sextilhas em redondilha maior**

Geração assistida de cordel para textos bíblicos, desenvolvida por [OBT Lab / Shema Bible Translation](https://shemabibletranslation.com).

## Architecture

The app uses a **three-agent pipeline** to transform biblical Prose Meaning Maps into formally correct cordel poetry (sextilhas in redondilha maior):

1. **Planejador** — reads Level 1 (Shape) → outputs structural plan, emotional arc, contrasts
2. **Vocabulário** — reads Level 2 (Scenes) → maps participants, places, objects to sertanejo vocabulary
3. **Compositor** — reads Level 3 (Propositions) + Agent 1 & 2 outputs → composes sextilhas

After composition, two additional phases are available:
- **Análise** — metric escansão (syllable verification), rhyme checking, fidelity assessment
- **Revisão** — targeted fixes for identified problems (up to 3 passes)

## Cordel Formal Rules

- **Sextilha**: 6 verses per strophe
- **Redondilha maior**: 7 metric syllables per verse
- **Rhyme scheme**: ABCBDB (verses 2, 4, 6 rhyme)
- **Vocabulary**: Sertanejo rural, concrete, oral

## Setup

```bash
# Clone
git clone https://github.com/shema-bible-translation/forja-de-cordel.git
cd forja-de-cordel

# Install dependencies
npm install

# Configure environment
cp .env.example .env.local
# Edit .env.local with your API keys

# Run locally
npm run dev
```

## Environment Variables

| Variable | Required | Description |
|---|---|---|
| `ANTHROPIC_API_KEY` | Yes | Claude API key for the three-agent pipeline |
| `ELEVENLABS_API_KEY` | No | ElevenLabs API key for audio generation |
| `ELEVENLABS_VOICE_ID` | No | ElevenLabs voice ID (configure a Brazilian voice) |

## Deploy to Vercel

1. Push to GitHub
2. Import the repo in [Vercel](https://vercel.com)
3. Add environment variables in Vercel project settings
4. Deploy

The API routes use `maxDuration = 120` which requires Vercel Pro or higher for the compose endpoint (3 sequential Claude calls).

## Input Format

The app expects a **Prose Meaning Map** generated by the Bible Meaning Maps app, containing:
- **Level 1 — The Shape**: structural overview, emotional register, contrasts
- **Level 2 — Scenes**: participants, places, objects, actions (2A–2E sections)
- **Level 3 — The Propositions**: semantic content that must appear in the cordel

## Tech Stack
